gogs 运行在后台后出现问题不会关😭

```
 $ nohup ./gogs web &
```
后台运行后可以使用

```
$ lsof -i:3000
COMMAND   PID USER   FD   TYPE DEVICE SIZE/OFF NODE NAME
gogs    17483  git    6u  IPv6 338433      0t0  TCP *:3000 (LISTEN)
```
然后就可以使用杀掉进程了，最后就是你放心的启动gogs吧😁

```
$ kill -9 17483 
```
